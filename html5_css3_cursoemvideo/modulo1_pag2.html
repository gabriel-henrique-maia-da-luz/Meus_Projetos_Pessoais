Imagens Dinâmicas

Tente abrir esse site em vários dispositivos diferentes ou aumente e diminua o tamanho da janela do navegador

Para fazer essas imagens dinâmicas, utilizamos a tag <picture> e várias tags <source> com diferentes condições de mídia.

Exemplo:

    

    <picture>
        <source media="(max-width: 750px)" srcset="imagens/imagem_p.png" type="image/png">
        <source media="(max-width: 1000px)" srcset="imagens/imagem_m.png" type="image/png">
        <img src="imagens/imagem_g.png" alt="Imagem flexível G">
    </picture>

Onde o <picture> é a tag principal que contém as diferentes fontes de imagem, e os <source> são as condições de mídia que determinam qual imagem será exibida.

Nesse caso, a imagem exibida dependerá do tamanho da janela do navegador.
Com menos de 750px, a imagem pequena (imagem_p.png) será exibida;
entre 750px e 1000px, a imagem média (imagem_m.png) será exibida;
e acima de 1000px, a imagem grande (imagem_g.png) será exibida.

Uma observação importante é que a ordem do tamanho sempre deve ser do menor para o maior.

Adicionando áudio

Para adicionar áudio em uma página HTML5, utilizamos a tag <audio>.

O atributo controls é utilizado para exibir os controles de reprodução do áudio, como play, pause e volume. Sem esse atributo, o áudio seria reproduzido automaticamente sem a possibilidade de controle pelo usuário.

O atributo loop é utilizado para repetir a reprodução do áudio automaticamente.

Adicionando vídeo

Para adicionar vídeo em uma página HTML5, utilizamos a tag <video>.

E para adicionar legendas, utilizamos a tag <track>.

E para adicionar descrições, utilizamos a tag <track> com o atributo kind="descriptions".

E para adicionar uma capa (thumbnail), utilizamos a tag <video> com o atributo poster.
