REGRAS BÁSICAS EM HTML


1. As tags HTML são escritas entre os sinais de menor e maior (< e >).

2. A maioria das tags tem uma tag de abertura e uma tag de fechamento, mas algumas são auto-fechadas.

3. O conteúdo de uma página HTML é colocado entre as tags de abertura e fechamento.

4. O código HTML é interpretado pelo navegador para exibir a página corretamente.

5. As tags HTML são case-sensitive, ou seja, <P> e <p> são diferentes.


As tags HTML mais usadas são:

<h1> a <h6> - Títulos

<p> - Parágrafo

<strong> - Negrito

<em> - Itálico

<br> - Quebra de linha



Parágrafos e quebras de linha

O elemento <p></p> é usado para criar parágrafos em HTML. Ele é um elemento de bloco, o que significa que ele ocupa toda a largura disponível e inicia uma nova linha após o seu fechamento.

O elemento <br> é usado para criar quebras de linha em HTML. Ele é um elemento vazio, o que significa que ele não tem uma tag de fechamento. Ele é usado para criar uma nova linha sem iniciar um novo parágrafo.

Formatos de imagens em HTML:


PNG, JPEG, GIF, SVG


O formato PNG (Portable Network Graphics) é amplamente utilizado para imagens com transparência e gráficos simples.

O formato JPEG (Joint Photographic Experts Group) é ideal para fotografias e imagens com muitos detalhes, pois oferece uma boa compressão sem perda significativa de qualidade.

O formato GIF (Graphics Interchange Format) é conhecido por suportar animações e é frequentemente usado para criar imagens animadas, embora tenha uma paleta de cores limitada.

O formato SVG (Scalable Vector Graphics) é um formato de imagem vetorial que é escalável sem perda de qualidade, tornando-o ideal para gráficos, logotipos e ícones.

Favicons

Favicons são pequenas imagens associadas a um site, exibidas na aba do navegador e em outros lugares onde o site é referenciado.

Basicamente, um favicon é um ícone pequeno que representa um website e é exibido na aba do navegador, na barra de endereços e em outros locais onde o site é acessado.

Ou seja, é o ícone que aparece na aba do navegador ao lado do título da página.

Para criar um favicon, você pode usar uma imagem quadrada, geralmente com dimensões de 16x16 pixels ou 32x32 pixels. O formato mais comum para favicons é o .ico, mas também é possível usar formatos como .png ou .svg.

Hierarquia de Títulos

Em HTML, existem seis níveis de títulos, que vão de <h1> a <h6>. O título <h1> é o mais importante e geralmente é usado para o título principal da página, enquanto os títulos <h2> a <h6> são usados para subtítulos e seções subsequentes.

A hierarquia de títulos é importante para a estruturação do conteúdo e para a acessibilidade, pois ajuda os leitores a entenderem a organização do conteúdo e facilita a navegação em dispositivos assistivos.

É recomendado usar os títulos de forma hierárquica, ou seja, começar com <h1> para o título principal e usar os títulos subsequentes (<h2>, <h3>, etc.) para organizar o conteúdo em seções e subseções.

Além disso, os títulos também são importantes para o SEO (Search Engine Optimization), pois os motores de busca usam os títulos para entenderem a estrutura e o conteúdo da página.

Por Exemplo:

Título Principal (h1)

🠒 Subtítulo 1 (h2)

🠒 Subtítulo 1.1 (h3)

🠒 Subtítulo 2 (h2)

🠒 Subtítulo 3 (h2)

🠒 Subtítulo 3.1 (h3)

🠒 Subtítulo 3.2 (h3)


Semântica em HTML5

A semântica em HTML5 refere-se ao uso de elementos que têm significado específico e descrevem a estrutura e o conteúdo do documento, tornando-o mais acessível e compreensível para usuários e motores de busca.


Formatação de Texto

Principais formatações de texto

Negrito/Destaque:

Para criar um texto em negrito em HTML, você pode usar a tag <strong> ou a tag <b>.

A tag <strong> é usada para indicar que o texto tem uma importância especial ou ênfase,
enquanto a tag <b> é usada apenas para estilizar o texto em negrito, sem transmitir nenhum significado adicional.

Italico/Ênfase:

Para criar um texto em itálico em HTML, você pode usar a tag <em> ou a tag <i>.

A tag <em> é usada para indicar que o texto tem uma ênfase especial,
enquanto a tag <i> é usada apenas para estilizar o texto em itálico, sem transmitir nenhum significado adicional.

Marcador de Texto

Para destacar um texto em HTML, você pode usar a tag <mark>, que é usada para marcar ou destacar um trecho de texto, geralmente com uma cor de fundo amarela.

Citações completas

Para criar uma citação completa em HTML, você pode usar a tag <blockquote>.

Segue a explicação dada pelo Prof.Willian Canellas sobre o que é um limite, na matéria de Calculo:

A ideia central por trás dos limites em matemática é a aproximação. Pense naquela xícara de café fervendo que você acabou de preparar. Ela está a 90°C, mas o ambiente ao redor está 30°C. O que acontece a seguir? A temperatura do café começa a diminuir, buscando um equilíbrio com o ambiente.
Nos primeiros minutos, essa queda pode ser quase linear, como se perdesse calor a uma velocidade constante. Mas, à medida que se aproxima dos 30°C, a mudança se torna mais lenta, tendendo a essa temperatura sem talvez nunca a atingir exatamente (ou levando um tempo infinito para isso). Esse "tender a" é a essência do limite.
Podemos imaginar uma função que descreve essa queda. Se quisermos saber quando o café atingirá (ou, mais precisamente, se aproximará infinitamente) dos 30°C, estamos procurando um limite.

As citações completas são úteis para destacar trechos de texto que são citados de outras fontes.

Link's para Download

Para criar um link para download em HTML, você pode usar a tag <a> com o atributo href, especificando o caminho do arquivo que deseja disponibilizar para download. Além disso, você pode usar o atributo download para sugerir que o navegador baixe o arquivo em vez de abri-lo.

Link's Internos

Links internos são links que apontam para outras partes do mesmo documento ou para outros documentos dentro do mesmo site. Eles são usados para facilitar a navegação e a organização do conteúdo em um site.

Para criar um link interno, você pode usar a tag <a> com o atributo href, especificando o destino do link usando um identificador (ID) ou um nome de âncora.

EXEMPLO: Ir para Imagens

Para criar um link interno para uma seção específica do mesmo documento, você pode atribuir um ID a essa seção usando o atributo id e, em seguida, usar esse ID como destino do link.
