REGRAS BÁSICAS EM HTML


1. As tags HTML são escritas entre os sinais de menor e maior (&alt; e >).

2. A maioria das tags tem uma tag de abertura e uma tag de fechamento, mas algumas são auto-fechadas.

3. O conteúdo de uma página HTML é colocado entre as tags de abertura e fechamento.

4. O código HTML é interpretado pelo navegador para exibir a página corretamente.

5. As tags HTML são case-sensitive, ou seja, <P> e <p> são diferentes.

As tags HTML mais usadas são:

<h1> a <h6> - Títulos

<p> - Parágrafo

<strong> - Negrito

<em> - Itálico

<br> - Quebra de linha



Parágrafos e quebras de linha

O elemento <p></p> é usado para criar parágrafos em HTML. Ele é um elemento de bloco, o que significa que ele ocupa toda a largura disponível e inicia uma nova linha após o seu fechamento.

O elemento <br> é usado para criar quebras de linha em HTML. Ele é um elemento vazio, o que significa que ele não tem uma tag de fechamento. Ele é usado para criar uma nova linha sem iniciar um novo parágrafo.
